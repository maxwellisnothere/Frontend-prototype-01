⚙️ ฝั่ง Backend (ระบบหลังบ้าน)

ระบบ Cache ราคา Steam (ป้องกันการโดนแบน): สร้างตัวดักจับราคาแบบ 5 นาที (Cache) เพื่อไม่ให้เซิร์ฟเวอร์โดน Steam บล็อกเวลาคนเข้าหน้าคลังแสงเยอะๆ

รองรับราคาทศนิยม: ถอดตัวปัดเศษทิ้งทั้งหมด ทำให้ระบบคำนวณราคาและแสดงผลทศนิยม 2 ตำแหน่งได้อย่างแม่นยำ

อัปเกรดระบบซื้อขาย (Buy Item): แก้ไขให้ Backend รับ listingId เพื่อหักเงินผู้ซื้อจริง สร้าง Order ลง Database และดึงชื่อคนซื้อ/คนขายมาบันทึกอย่างถูกต้อง

เพิ่ม API ประวัติการทำรายการ: สร้าง Endpoint /orders/buy และ /orders/sell ให้แอปดึงประวัติจาก Database ของจริง (แทนการเซฟลงความจำมือถือ)

📱 ฝั่ง Frontend (แอปพลิเคชัน)

แก้บั๊ก Login & Loading: เปลี่ยนช่อง Developer Login ให้ใส่แค่ SteamID และแก้ปัญหาปุ่มหมุนค้าง (Loading Spinner) ทั้งในหน้า Login, เติมเงิน และถอนเงิน

Inventory & Item Detail สุดลื่นไหล: เชื่อม API ให้ดึงราคาสดจาก Steam (Real-time) พร้อมแก้บั๊กราคากลายเป็น 0 บาทตอนกดเข้าไปดูรายละเอียด

StoreScreen (หน้าร้านค้าของจริง): ล้างบางไอเทมจำลองทิ้งทั้งหมด เปลี่ยนให้หน้านี้ไปดึงข้อมูลเฉพาะ "ไอเทมที่มีคนกดวางขายแล้วจริงๆ (Listings)" มาแสดงเท่านั้น

Checkout & Payment หักเงินจริง: แก้ให้ส่งรหัสสินค้า (listingId) ไปหา Backend อย่างถูกต้อง พอกดซื้อสำเร็จยอดเงินจะลดลง และไอเทมจะหายไปจากตลาดทันที

รื้อระบบประวัติการซื้อ-ขายใหม่หมด:

หน้า Buy History: เปลี่ยนมาดึงข้อมูลจาก Database สดๆ พร้อมบอกสถานะ (รอรับของ, สำเร็จ)

สร้างหน้า Sale History (ใหม่!): สำหรับพ่อค้าแม่ค้าไว้ดูว่าของตัวเองชิ้นไหนขายออกไปแล้วบ้าง

เพิ่มระบบ กรอก Trade Offer ID: ในหน้า Order Detail ถ้าเราเป็นคนขาย ระบบจะโชว์ช่องให้เรากรอกเลข Trade Offer เพื่อยืนยันการจัดส่งของให้คนซื้อ
